
Variante pentru algoritm:

Varianta 1:

1.  Se alege în ordine sau aleatoriu câte un student din lista și se face asignarea în funcție de ordinea preferințelor, având în vedere capacitatea laboratoarelor.
2.  Dacă slotul preferat de student este deja ocupat la capacitate maximă, se trece la următoarea opțiune din lista preferințelor.
3.  Acest proces se repetă până când toți studenții sunt asignați.

Varianta 2:

1.  Se alege în ordine sau aleatoriu câte un student din lista și se face asignarea în funcție de ordinea preferințelor, fără a lua în considerare capacitatea laboratoarelor.
2.  După finalizarea primei etape, se verifică pe rând fiecare slot și se constată dacă numărul de studenți depășește capacitatea laboratoarelor.
3.  Se alege în mod aleatoriu sau în ordine un slot care depășește capacitatea, iar din acest slot se alege la întâmplare sau în ordine un student care este reasignat la următoarea opțiune, distrugând astfel anumite "perfect matchings" deja existente.
4.  Acest proces se repetă până când toți studenții sunt asignați.

Varianta 3 - preferată:

1.  Se alege în mod aleator câte un student din listă și se face asignarea în funcție de ordinea preferințelor, fără a lua în considerare capacitatea laboratoarelor.
2.  După finalizarea primei etape, se verifică pe rând fiecare slot și se constată dacă numărul de studenți depășește capacitatea laboratoarelor.
3.  Se alege aleatoriu un slot care depășește capacitatea, iar din acest slot se alege aleatoriu un student care este reasignat la următoarea opțiune în funcție de capacitatea slotului respectiv, protejând astfel "perfect matchings" existente.
4.  Acest proces se repetă până când toți studenții sunt asignați.
